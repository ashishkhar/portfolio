Cybersecurity / Data Visualization

GitHub Demo

Project Overview

The STIX2.1 Visualization Tool is an interactive web application for security analysts to visualize and analyze STIX (Structured Threat Information eXpression) data. This tool transforms complex threat intelligence expressed in STIX 2.1 format into intuitive graph visualizations, enabling analysts to better understand threat actor tactics, techniques, and relationships between cyber threat entities.

Key Features

  • Interactive Graph Visualization

    Dynamic force-directed graph layout with zoom, pan, and focus capabilities

  • Full STIX 2.1 Support

    Comprehensive implementation of all STIX 2.1 domain objects and relationship types

  • Advanced Filtering

    Multi-level filtering by object types, relationships, time range, and confidence levels

  • Pattern Analysis

    Automated identification of common attack patterns and relationship clusters

  • Temporal Analysis

    Timeline view showing evolution of threat activities and relationships over time

  • Export & Collaboration

    Export visualizations as images or interactive HTML, with annotation and sharing capabilities

Architecture

The system follows a microservices architecture with these key components:

  1. STIX Parser Service: Validates and processes STIX 2.1 JSON documents
  2. Graph Database: Stores parsed STIX objects and relationships in optimized graph structure
  3. Analytics Engine: Performs pattern identification and relationship analysis
  4. Visualization Engine: Renders interactive graph visualizations using D3.js
  5. Export Service: Generates sharable visualization outputs in various formats
  6. Authentication Layer: Manages user access and secures sensitive threat intelligence

Challenges & Solutions

Graph Performance

Challenge: Maintaining interactive performance with large STIX datasets (1000+ nodes)

Solution: Implemented WebGL-based rendering for large graphs and developed level-of-detail techniques to simplify visualization at different zoom levels

Complex Relationships

Challenge: Clearly visualizing complex many-to-many relationships between STIX objects

Solution: Developed custom edge bundling algorithm and contextual highlighting to emphasize relevant relationships

STIX Specification Evolution

Challenge: Adapting to evolving STIX specification and maintaining backward compatibility

Solution: Created adapter layer for different STIX versions and implemented validation pipeline to ensure spec compliance
